## 2026-05-15

### Domain Routing

- Investigated the remaining `www.kishu.com` routing problem after the main `kishu.com` zone was moved under current cleanup control.
- Confirmed `kishu.com` and forwarding/test domains can be routed away from old content, while `www.kishu.com` was still being handled outside the current Cloudflare zone path by an old Webflow/Cloudflare-for-SaaS style binding.
- Built and deployed a dedicated `kishu-inu-domain-forwarder` Cloudflare Worker as a routing diagnostic and fallback forwarder for Kishu-related forwarding-only domains.
- Used a simple Kishu loading page and response headers to tell whether a request was reaching the current Worker or still being served by stale legacy infrastructure.
- Moved DNS control for `kishu.com` back to GoDaddy to use registrar-level forwarding while the stale `www.kishu.com` binding is resolved.
- Temporary domain state: `kishu.com` forwards toward `kishu.finance` / `www.kishu.finance`; `www.kishu.com` is no longer being intentionally maintained as the old cached Webflow site and may remain blank or inconsistent while forwarding and hostname cleanup settle.

### Notes

- No owner Safe or multisig access changed.
- No legacy wallet access changed.
- The immediate priority was reducing exposure to outdated or unsafe legacy links while keeping current public assets separated from abandoned infrastructure.
